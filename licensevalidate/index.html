Apa Itu License Validate

Aplikasi ini merupakan sebuah program yang memvalidasi licensi dari inputan pengguna (software desktop). Dimana aplikasi ini ditunjukkan untuk latihan saya sebagi pembuat untuk mempraktikkan, secara sederhana bagaimana aplikasi seperti Adobe, Corel Draw dan sebagainya bekerja dalam hal aktivasi licensi. Project ini terdiri dari 2 bagian yakni :

1. License validate server

ini merupakan server web untuk memvalidasi request, mengenerate license key. dibuat dengan java springboot web, dan transferring data menggunakan json format.
Link : license validate server

2. License validate client (desktop)

ini merupakan aplikasi yang mengambil data input dari user dan mengirimkan ke server, ini menggunakan javafx.
Link : license validate client

Tujuan Aplikasi

Tujuan aplikasi ini adalah untuk melatih saya sebagai pembuat dalam hal pembuatan program dengan bahasa pemrograman java.
